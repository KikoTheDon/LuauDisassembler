# LuauDisassembler
A disassembler for Roblox's new bytecode format. This is still in development and does not support all op-codes.

# Supported op codes

- [x] OGETENV / GETGLOBAL
- [x] LOADNUMBER
- [x] LOADK
- [x] MOVE
